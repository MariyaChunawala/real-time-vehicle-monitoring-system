# Frontend — Real-Time Vehicle Monitoring System

This is the React + Vite frontend for the Real-Time Vehicle Monitoring System (TypeScript).

## Prerequisites
- Node.js v18+ (or the LTS you prefer)
- npm

## Install

```bash
cd frontend
npm install
```

## Development

```bash
npm run dev
```

Open `http://localhost:5173` (Vite default) in your browser.

## Build & Preview

```bash
npm run build
npm run preview
```

## Useful scripts
- `npm run dev` — Start Vite dev server
- `npm run build` — Compile TypeScript and build production assets
- `npm run lint` — Run ESLint

## Theming
This project uses CSS variables defined in `src/App.css`. Toggle `data-theme="light"` or `data-theme="dark"` on the document root to switch themes.

## Troubleshooting
- If the frontend cannot reach the backend, verify the backend server is running on the expected port.
